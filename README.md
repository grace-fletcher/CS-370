# CS-370: Current and Emerging Trends in Computer Science
This repository includes work done for CS 370: Current and Emerging Trends in Computer Science


For the final project of this course, I was given starter code for an AI pirate agent. The goal of this agent was to successfully learn how to traverse and complete a maze problem. From the starter code, I successfully implemented a reinforcement learning algorithm to train the agent on how to complete the maze. I've also uploaded my final project report document, as well as other course assignment documents that are related to the concept of reinforcement learning.


## What do computer scientists do and why does it matter?
Computer scientists implement problem sovling skills with programmable technology to solve modern day issues. They are more important now than ever with the current trends of AI and machine learning in the modern day tech field.

## How do I approach a problem as a computer scientist?
To approach a probelm as a computer scientist, you must start by thinking logically and algorithmically. Specifically, you must use decomposition and abstraction methods to break a problem down and focus only on what needs to be fixed to find a solution.

## What are my ethical responsibilities to the end user and the organization?
As a developer, it is your ethical responsibility to provide effective and quality service while also being transparent in your methods to both the end user and the organization you work for. In short, you must disclose any and all drawbacks associated with your solution to a problem while also keeping up with coding best practices in your work.
